# Project 2 - Data Classification Using AI

## Objective
To build a machine learning classification model using the Iris Dataset.

## Dataset
Iris Dataset from Scikit-learn.

## Algorithm Used
Decision Tree Classifier.

## Steps Performed
1. Imported required libraries.
2. Loaded the Iris dataset.
3. Explored the dataset.
4. Split data into training and testing sets.
5. Trained a Decision Tree model.
6. Predicted test data labels.
7. Evaluated model accuracy.

## Result
The model achieved high classification accuracy on the test dataset.
